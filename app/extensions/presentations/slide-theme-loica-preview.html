Body copy

This is a paragraph at 36px base, line-height 1.45. The theme uses italics sparingly, bold at 600, and inline code with a faint backplate. Links carry a soft underline: loica.criti.ca.

Small text drops to 0.7em and dims for asides, captions, and timestamps.

Lists

  • First-level items use a disc bullet
  • Generous line spacing keeps long lists readable
    • Second level switches to an em-dash for visual variety
    • Stays quiet so it doesn't compete with the text
  • Items wrap cleanly on long lines without crowding adjacent items
  1. Ordered lists use the default numerals
  2. Same spacing rules apply

Knobs to design against

The theme exposes these CSS custom properties on .reveal:

--r-background-color
--r-main-color           /* body text */
--r-heading-color
--r-link-color
--r-link-color-hover
--r-main-font            /* Geist Sans + fallbacks */
--r-heading-font
--r-code-font            /* Geist Mono + fallbacks */
--r-main-font-size       /* 36px base */

Override any of these in ./slide-theme-loica.css, refresh, and iterate. Reveal core CSS handles layout/transitions; this file owns the look only.
